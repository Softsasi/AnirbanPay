# [AnirbanPay](https://Anirbanpay.com) — Payment Automation Platform

AnirbanPay is a self-hosted, open-source payment automation platform by QubePlug.  
Deploy it on your own server, customize your workflow, install plugins and themes, and stay fully in control — with no monthly fees.  
Visit https://Anirbanpay.com/ for more information.

## 🚀 Features

- Self-hosted deployment (no vendor lock-in)
- Plugin-based architecture for gateways, modules, and tools
- Theme support for admin dashboard branding
- Multiple payment gateways via plugins (Stripe, bKash, Nagad, etc.)
- Automatic transaction verification
- Unlimited transactions (server capacity dependent)
- Integrations with WooCommerce, WHMCS, WordPress, and more

## ⚙️ Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AnirbanPay/AnirbanPay.git
   ```

2. Prepare your environment:
   - PHP
   - Database (MySQL/MariaDB)
   - Web server (Apache/Nginx)

3. Upload files to your server  
4. Configure database credentials and environment  
5. Access the admin panel and install plugins
